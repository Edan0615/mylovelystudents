00. 核心串接：三種內建方法

01 行內式 (Inline)

直接將樣式或腳本寫在 HTML 標籤的屬性中。適合快速測試或單一元素的微調。

<!-- CSS Inline -->
<div style="color: red;">...</div>
<!-- JS Inline -->
<button onclick="alert('Hi!')">...</button>

02 內部區塊 (Internal)

在 HTML 檔案中開闢專屬區塊。CSS 放 <head>，JS 通常放 <body> 底部。

<style> /* 放於 head */
body { background: #eee; }
</style>
<script> /* 放於 body 底部 */
console.log("Loaded!");
</script>

03 外部導引連結 (External)

最專業的方法。將代碼獨立存為 .css 或 .js 檔案，實現結構與樣式的完美分離。

<!-- CSS Link (in Head) -->
<link rel="stylesheet" href="style.css">
<!-- JS Script (in Body) -->
<script src="app.js"></script>
最佳實務提點 (Best Practice)
為了網頁讀取速度與維護性，大多數專案應優先使用第一種與第三種方法。
CSS: in <head>
JS: before </body>

01. 標籤結構與語法

開標籤與閉標籤

在 HTML 中，大多數的標籤都是成對出現的。這就像是「括號」一樣，告訴瀏覽器這個元素的範圍從哪裡開始，到哪裡結束。

  • 開標籤 (Opening Tag): <tagname> - 標示內容的開始。
  • 閉標籤 (Closing Tag): </tagname> - 標示內容的結束（多了一個斜線）。
  • 有些標籤不需要閉標籤，稱為「自閉合標籤」，例如 <img> 或 <input>。
syntax_example.html
<p>這是一段開頭標籤。
內容放在中間...
</p>這是結束標籤。

02. 常用排版標籤

標題與段落 (H & P)

用於定義網頁的結構與內容。從 H1 (最重要) 到 H6 (最次要)；而 <p> 則代表段落文字。

H1 最重要標題

這是一個被 P 標籤包覆的段落，通常會自動換行並與其他元素保持間距。

H2 次級標題

H6 最小標題樣式
<h1>主標題</h1>
<p>內容段落文字...</p>
<h2>次級標題</h2>
<h6>最小標題</h6>

行內元素 (Span)

INLINE

<span> 是行內元素。它的寬度僅取決於內容，不會自動換行，適合用來修飾段落中的局部文字。

這是一段 重點文字 會留在原行。 

<p>
  這是 <span class="highlight">行內元素</span> 範例。
</p>

區塊元素 (Div)

BLOCK

<div> 是區塊元素。它預設會橫跨整行寬度 (100%)，即使內容很少也會換行，適合做為佈局容器。

Div A (佔據全寬)
Div B (自動換行) 
<div class="block-element">
  我是區塊元素，我會換行。
</div>

03. 表單與輸入 (Form)

? 如何與使用者互動？

<form> 標籤負責宣告一個表單區塊。而內部的 <input> 則提供不同的輸入方式，透過 type 屬性來決定它是哪種功能。

Text / Password
最基礎的文字與密碼輸入框。
Date / Number
受限格式的日期選取器或數字調整。
Checkbox / Radio
多選一或複選的多功能開關。

Live Interactive Demo 

<form>
  <input type="text">
  <input type="password">
  <input type="date">
  <input type="checkbox">
  <button>Submit</button>
</form>

04. 多媒體呈現 (Multimedia)

Picture / Image

不僅可以用 <img>，還能用 <picture> 根據螢幕大小提供不同尺寸的圖片，優化效能。

<picture><source><img></picture> 
<picture>
  <source media="(min-width: 650px)" srcset="...">
  <img src="..." alt="響應式圖片">
</picture>

Audio / Video

現代瀏覽器支援原生的多媒體播放器，只需要指定路徑與 controls 屬性即可。 

<!-- 音訊 -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<!-- 影片 -->
<video controls poster="cover.jpg">
  <source src="video.mp4" type="video/mp4">
</video>

CSS 核心基礎

如果說 HTML 是網頁的骨架，那麼 CSS 就是它的衣服與靈魂。讓我們來看看如何運用 CSS 為網頁穿上層次感。

1 選擇器 (Selectors)

button / p / div ELEMENT

直接用標籤名，會套用到全網頁該類型的元素。

.className CLASS

用點（.）來選取所有具有該類別名的元素，適合重複使用。

#elementID ID

用井字號（#）選取唯一的 ID，每個 ID 在同一頁面僅限出現一次。

2 語法結構 (Syntax)

.title-text {
color: #0ea5e9; // 字體顏色
background-color: white; // 背景顏色
display: flex; // 顯示模式
transition: 0.3s ease; // 動畫過渡
}

Flexbox 佈局彈性

Flex 核心屬性
  • display: flex; - 啟動彈性盒子。
  • justify-content: center; - 水平居中。
  • align-items: center; - 垂直居中。
  • gap: 1rem; - 元素間距。

12 欄位系統規則 (Grid System)

如同 Bootstrap，Flex 常用於建構 12 欄位響應式網格。

03. jQuery 互動魔法箱

Introduction to Library

為什麼選擇 jQuery？

jQuery 是 JavaScript 的「快速開發工具箱」。它將複雜的瀏覽器操縱簡化為幾行直覺的代碼，讓開發者能以 **「Write Less, Do More」** 的初衷打造生動的網頁特效。

快速上手
跨瀏覽器相容

Code 對決：簡約的極致

Native JS (傳統做法) 
document.getElementById("btn")
  .addEventListener("click", function() {
    this.style.display = "none";
  });
VS
jQuery (魔法寫法)
$("#btn").click(function() {
  $(this).hide();
});

精準選取 (Sizzle)

使用 $() 符號配合 CSS 選取器，就能瞬間控制網頁上的任何元素，不再需要冗長的 JS 語法。

事件監聽 (.on)

點擊、滾動、輸入——所有使用者互動都能透過 .on() 輕鬆捕捉並觸發後續反應。

內建特效 (.fadeOut)

想要淡入、滑動或切換？內建的動畫方法讓你只需一行代碼就能完成專業級的視覺回饋。

CDN 核心概念

無需下載檔案，直接透過「內容傳遞網路」引用。它能讓你從最近的伺服器抓取代碼，優化載入速度。

<script src="https://...jquery.min.js"></script>

踏向未來的墊腳石

"Learning the ladder, reaching the stars."

雖然現在主流是 Vue 或 React 等框架，但 jQuery 建立的 **「操作 DOM」** 與 **「處理事件」** 的思維依然是前端不可或缺的底功。學會了它，你將能更快理解現代框架在底層到底幫你解決了什麼。
