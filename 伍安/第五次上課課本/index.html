Strategy

專案規模與框架選擇

大型專案 Vue.js / React
  • • 組件化開發： 像樂高一樣搭建，易於維護。
  • • 數據響應式： 資料變動，畫面自動更新。
  • • 虛擬 DOM： 適合高性能的大規模運算。
  • ※ 適合商城、管理系統、大型跨國網站。
中小型專案 jQuery / Vanilla JS
  • • 開發速度極快： 引入即用，無需複雜安裝。
  • • 操作直觀： 直接控制 DOM 元素。
  • • 專注交互： 適合寫簡單的動畫與表單提交。
  • ※ 適合行銷活動頁、形象網站、單頁 Prototype。

The Battle

函式庫與框架的正面對決

🎨 Tailwind vs Bootstrap

Bootstrap (組件導向)

提供現成的按鈕、導航欄樣式。快，但容易「撞衫」，且想修改細節時會被預設 CSS 絆住。

Tailwind CSS (原子化導向)

不提供組件，只提供「樣式碎片」。你需要自己組合出的 components，完全自由，設計質感最上限極高。

⚡ jQuery vs Vanilla JavaScript

// jQuery 寫法 (超短!)
$('.btn').click(() => $('.box').fadeIn());
// 原生 JS 寫法 (邏輯嚴謹)
document.querySelector('.btn').addEventListener('click', () => {
  const box = document.querySelector('.box');
  box.style.display = 'block';
  box.style.opacity = 1;
});

結論：jQuery 大幅降低了「選擇器」與「動畫」的複雜度，對初學者與小專案極度友善。
