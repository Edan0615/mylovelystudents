💡 程式開發心法

「在動手寫程式之前，先武裝你的大腦」

K

K.I.S.S.

Keep It Simple and Stupid

保持簡單明瞭。最好的程式碼是連新同學看了都能秒懂的結構。不要過度設計，能用簡單邏輯解決的問題，就不要寫得太深奧。

🎯 最簡單的解法，通常就是最好的解法
D

D.R.Y.

Don't Repeat Yourself

不要重複你自己。如果同樣的代碼寫了兩次以上，就應該將它「模組化」。

// ❌ Bad: 重複寫三次
print("Hello A")
print("Hello B")
print("Hello C")
// ✅ Good: 封裝成函式
function say(name) {
  print("Hello " + name)
}
⚡ 懶惰是工程師的美德：我們只寫一次，然後重複使用
P

殺雞焉用牛刀

The Principle of Least Power

選擇最合適的工具。每種語言都有擅長的領域。就像廚房裡的西瓜刀與水果刀，雖然功能重疊，但各有優勢。高手會用最輕量、最快速的技術達成目標。

🛠️ 本課實踐：用 jQuery 簡化 JS、用 Tailwind 加速佈局、透過 CDN 輕量導入。

🏗️ 網頁三本柱

「HTML 為骨、CSS 為皮、JavaScript 為肌」

HTML

網頁的骨架

定義內容的結構，如標題、段落與按鈕。就像是房子的主體結構。

CSS

網頁的皮囊

負責網頁的顏色、排版與動畫。就像是房子的裝潢與設計美感。

JavaScript

網頁的肌肉

提供網頁的互動行為與動態效果。就像是房子的電力與自動化系統。

🧩 框架與函式庫

Bootstrap

它能大幅簡化 CSS 的撰寫。核心理念是「封裝元件」。讓我們能快速拼湊出專業級的介面。

Navbar 實作對比
Vanilla HTML/CSS
<nav class="my-nav">
  <div class="container">
    ...
</nav>

.my-nav {
  display: flex;
  padding: 1rem;
  background: #fff;
  ... (需要 20+ 行 CSS)
}
Bootstrap Magic
<nav class="navbar bg-light">
  <div class="container">
    ...
</nav>

// 免寫額外 CSS！
直接用內建 Class 搞定
🚀 一行 Class 勝過二十行 CSS，這就是框架的力量
jQ

jQuery

簡化 JavaScript 的函式庫。核心理念是 "Write less, do more"。讓我們更輕鬆地操作元素。

// 如何隱藏一張圖片？
原生 JavaScript (Vanilla JS)
document.querySelector('.img').style.display = 'none';
⚡
jQuery 極簡法
$('.img').hide();

🌐 網路基礎知識

「網頁是如何從雲端傳到你的手機裡的？」

📱

用戶端 (Client)

你的手機或電腦。當你輸入網址，就是在發出「請求 (Request)」。

☁️

伺服器 (Server)

遠端的電腦。收到請求後，把網頁檔案打包「回應 (Response)」給你。

實戰演練：從零構建一套計算機

一步步觀看 HTML (骨)、CSS (皮)、JavaScript (肌) 如何協作

Phase 01: HTML

建立內容骨架

我們首先用 HTML 定義內容。這時候完全沒有排版與按鈕樣式，只有一堆文字和按鈕散落在頁面上。

<!-- 顯示器 -->
<div class="display">0</div>

<!-- 數字按鈕 -->
<button>1</button> <button>2</button> ...
Phase 02: CSS

賦予視覺皮囊

加入 CSS 後，我們定義了顏色、間距與 Grid 網格佈局。這讓網頁看起來像一個現代化的工具設計。

.calculator { display: grid; }
.display { background: #f8fafc; }
button {
  padding: 1.5rem;
  border-radius: 1rem;
}
Phase 03: JavaScript

注入互動肌肉

最後加入 JavaScript。當按鈕被點擊，程式會捕獲數值並即時反映在畫面上。這時候計算機才真正「活」了過來！

const buttons = document.querySelectorAll('.btn');

buttons.forEach(btn => {
  btn.addEventListener('click', () => {
    let num = btn.innerText;
    document.querySelector('#screen').innerText = num;
  });
});
