The Soul of Web

賦予網頁生命：
JavaScript 全解析

網頁開發是由三根巨大的柱子支撐起來的：HTML 定義結構，CSS 決定美感，而 **JavaScript** 則是賦予靈魂與肌肉的關鍵。

Module 01

JavaScript 的定位：三本柱

HTML (骨架)

內容的結構。標題、文字、圖片的位置。

CSS (皮囊)

視覺的美感。顏色、字體、間距、排版。

JavaScript (靈魂/肌肉)

網頁的動作。點擊反應、資料運算、動態交互。

「如果沒有 JavaScript，網頁就像一張精美的海報；有了它，網頁才真正變成了一個應用程式。」

Module 02

為什麼我們需要 jQuery？

原生的 JavaScript (Vanilla JS) 在操作網頁元素時，語法往往非常繁瑣且在不同瀏覽器間存在相容性問題。

  • ✓

    語法簡潔

    Write Less, Do More. 一行代碼抵十行。

  • ✓

    跨瀏覽器相容

    不用擔心 Safari 或 Chrome 的微小差異。

代碼對比
// 原生 JavaScript
document.getElementById("btn").addEventListener("click", function() {
  document.querySelector(".box").style.display = "none";
});

// jQuery 寫法
$("#btn").on("click", function() {
  $(".box").fadeOut();
});
Module 03

JavaScript 學習的三大區塊

⌨️

1. 基礎語法 (Basic Syntax)

變數、迴圈、判斷式。這是所有程式語言共通的邏輯基石。

🏗️

2. DOM 操作 (Manipulation)

抓取 HTML 標籤，動態修改文字、顏色或結構。這就是 jQuery 的強項。

🌐

3. 資料傳輸 (Ajax/Axios)

向伺服器請求資料（如天氣、股市、會員資訊）並顯示在頁面上。

Module 04

核心語法詳解 (Syntax Deep Dive)

📦

變數 (Variables)

用來存放資料的「箱子」。現代開發推薦使用 `let` 與 `const`。

let age = 20; // 會變動的資料
const name = "Pinhua"; // 不會變動的常數
🚦

判斷式 (If/Else)

根據條件決定程式的走向。如果...就...，否則就...

if (score >= 60) {
  console.log("及格！");
} else {
  console.log("不及格");
}
🎛️

切換開關 (Switch)

當條件很多時（如：週一到週日），用 Switch 會比 If/Else 更整潔。

switch (day) {
  case "Monday": console.log("上課"); break;
  case "Sunday": console.log("休息"); break;
  default: console.log("努力中");
}
🔄

迴圈 (Loops)

重複執行一段代碼。最常用的是 `for` 與 `while`。

For 迴圈 (已知次數)
for (let i = 0; i < 5; i++) {
  console.log("For 第" + i + "次");
}
While 迴圈 (未知次數/條件成立就跑)
let count = 0;
while (count < 5) {
  console.log("While 第" + count + "次");
  count++;
}
🛑

流程控制 (Break / Continue)

Break (直接跳出)
for (let i=0; i<10; i++) {
  if (i === 5) break; // 停在 5
  console.log(i);
}
Continue (跳過這一次)
for (let i=0; i<10; i++) {
  if (i === 5) continue; // 略過 5
  console.log(i);
}
📚

陣列 (Arrays)

一組有序的資料集合。就像一個有很多隔層的抽屜。

let fruits = ["蘋果", "香蕉", "橘子"];
console.log(fruits[0]); // 印出 蘋果
🛠️

函式 (Functions)

將一段邏輯打包起來，需要時直接呼叫（Call）。這就是「模組化」。

function sayHi(name) {
  return "你好, " + name;
}
sayHi("蘇品樺"); // 執行！

🚀 小結

這些基礎語法是所有邏輯的靈魂。我們學這些不是為了考試，而是為了能像拼圖一樣，把這些邏輯拼湊成有趣的網頁功能。
