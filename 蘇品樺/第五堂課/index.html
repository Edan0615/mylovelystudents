DOM 元素選取器大全

document.getElementById()

最快、最基本

透過 HTML 標籤的 id 屬性來尋找元素。因為在一個頁面中 ID 必須是唯一的，所以這個方法只會回傳一個元素。

// HTML: <div id="my-title">標題</div>
const title = document.getElementById('my-title');

getElementsByClassName()

透過 class 名稱來尋找，會回傳所有符合的元素集合 (HTMLCollection)。

const items = document.getElementsByClassName('list-item');

getElementsByTagName()

透過 HTML 標籤名稱 (如 div, p, a) 來尋找，同樣回傳元素集合。

const links = document.getElementsByTagName('a');

document.querySelector()

⭐ 最強大推薦

現代開發最常用的方法！你可以直接傳入 CSS 選擇器，它會回傳「第一個」找到的元素。非常直覺且強大。

// 找 ID
document.querySelector('#my-title');

// 找 Class
document.querySelector('.list-item');

// 找複雜結構 (類別為 container 下的第一個 a 標籤)
document.querySelector('.container a');

document.querySelectorAll()

如果你的 CSS 選擇器會對應到多個元素，且你想要一次把他們全部抓出來（回傳 NodeList 陣列），就使用這個。

const allItems = document.querySelectorAll('.list-item');

讓網頁活起來：事件監聽

addEventListener()

找到元素後，下一步就是要幫它加上「耳朵」，讓它能隨時監聽使用者的動作（如點擊、輸入等），並做出相對應的反應。

語法結構

element.addEventListener('事件名稱', function() {
    // 當事件發生時，要執行的程式碼
});
