關於這個教學專案

本專案旨在幫助初學者理解網頁的基本構成。

我們使用了 Bootstrap Grid (網格系統) 來處理排版，這也是目前 RWD (響應式網頁設計) 的核心概念。無論是在手機還是桌機上，網頁都能完美呈現。

  • 語義化 HTML 標籤
  • 直觀的 CSS 命名
  • 簡單的 jQuery 事件處理
